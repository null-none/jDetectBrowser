# jDetectBrowser
Jquery plugin for detect browser

## Install

```bash
bower install jDetectBrowser
```

## Example

```html
...
  <script src="jDetectBrowser/jDetectBrowser.js"></script>
...

```

```js
$(".os").browserDetect()['OS'];
$(".version").browserDetect()['version'];
$(".browser").browserDetect()['browser'];
```
